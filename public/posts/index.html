WireGuard Dead Switch on Raspbian

WireGuard on Raspbian as Dead Switch Modify rc.local in /etc. Enter those lines after fi. piuk is a name of the WireGuard network interface. sleep 15 sudo iptables -t nat -A POSTROUTING -o piuk -j MASQUERADE sudo iptables -A FORWARD -i piuk -o eth0 -m state --state RELATEDESTABLISHED -j ACCEPT sudo iptables -A FORWARD -i eth0 -o piuk -j ACCEPT exit 0
